# ExpressJS

## SetUp

#### 1. Install Node.js
#### 2. Create Folder ( example :- backend ) => _project name should not contain any capital letter._
#### 3. cd Folder
#### 4. Create file app.js/index.js
#### 5.  ( in terminal ) => this cmd generate package.json file
    npm init -y
   
#### 6. Install express framework
    npm i express
